CSS: Estilizando sua Página Web

CSS, ou Cascading Style Sheets, é a linguagem de estilo que permite dar vida ao seu HTML. Neste artigo, abordamos os conceitos básicos do CSS, incluindo seletores, propriedades de estilo e o modelo de caixa.


/* Exemplo de código CSS */
body {
    font-family: 'Arial', sans-serif;
    background-color: #f0f0f0;
    margin: 0;
    padding: 0;
}

header {
    background-color: #333;
    color: #fff;
    padding: 20px;
    text-align: center;
}

section {
    margin: 20px;
    padding: 20px;
    background-color: #fff;
    box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
}

h1, h2 {
    color: #333;
}

p {
    color: #666;
}
